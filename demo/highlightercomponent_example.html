Text Highlighter Examples

Image Highlighter Example

This example demonstrates the highlighter toolbar and highlight mode logic on an image. Click the pin to activate highlight mode. The toolbar will appear on hover or after activation. See the README for usage details.

Basic Example

This is a basic example of the highlighter component. Look for the pin icon in the bottom-right corner. Try clicking it to activate the highlighter.

Once activated, you can select text in this box and highlight it. The pin will stay active until you click it again to deactivate the highlighter.

Custom Colors

This example uses custom colors for the highlights and buttons. The highlighting color is blue-themed instead of the default yellow.

You can customize the colors and other properties through the JavaScript API or CSS variables. This makes it easy to match your website's theme.

Mobile-Friendly Example

This example is optimized for mobile devices. The pin is slightly larger for better touch targets, and the highlighting behavior is adapted for touch screens.

Try using this on your mobile device - the pin should be easy to tap, and text selection should work smoothly.
