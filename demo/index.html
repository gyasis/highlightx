Highlighter Decorator Demo

Instructions

  1. Look for the pin icon ( ) in the corner of each content box
  2. Hover over the pin to see it animate
  3. Click the pin to activate highlighting mode
  4. Select text and use the highlight button that appears
  5. Click the pin again to deactivate highlighting

Default Configuration

This example uses the default highlighter configuration with green theme. Try selecting some text after activating the highlighter. Notice how the superscript numbers appear neatly above the highlighted text.

The numbers serve as reference points and can be clicked to navigate between highlights. They're designed to be unobtrusive while maintaining readability.

Custom Colors

This section demonstrates custom colors for both the highlights and the reference numbers. The blue theme provides a different visual style while maintaining the same functionality.

You can customize various aspects including the highlight color, number color, and pin appearance to match your website's design.

Advanced Features

This example shows advanced features like notes and navigation. After highlighting text, you can add notes, use keyboard shortcuts, and navigate between highlights using the controls or by clicking the reference numbers.

The sidebar provides an overview of all highlights with their associated metadata and notes. Try the search feature to quickly find specific highlights.
